Vuforia
Popular library for cross-platform dev:
- Vuforia HoloLens sample in Unity
- Vuforia works with the Microsoft MR Toolkit - Issues can be resolved

Vuforia provides object sensing capabilities not included in Microsoft Mixed Reality Toolkit. Vuforia also works with iOS and Android.

"Vuforia brings an important capability to HoloLens - the power to connect AR experiences to specific images and objects in the environment. You can use this capability to overlay guided step by step instructions on top of machinery or to add digital features to a physical product." - Microsoft

"Vuforia 7 introduces Model Targets, a new feature for recognizing objects by shape using existing 3D models. With Model Targets, content can be attached to objects such as automobiles and appliances, as well as industrial equipment and machinery, enabling a new class of AR content that can replace traditional user manuals and technical service instructions." - Vuforia
